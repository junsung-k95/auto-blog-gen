## 작성 원칙

- **누적**: 뒷 단계는 앞 단계 결정을 뒤집지 않는다. 바꿔야 하면 앞 문서를 먼저 고친다.
- **추적 가능**: 기능 ID(`1.1`, `2.8` 등)는 단계 간 일관 유지.
- **MVP 우선**: ⭐로 표기된 항목이 v1 범위. ⚪🔵는 우선순위 백로그.

## 빠른 요약

- **타겟**: N잡 블로거 (월 100만 원 자동 수익 목표)
- **루프**: 발굴 → 작성 → 최적화 → 발행 → 추적 → 학습
- **현재 구현**: 작성 · 일부 트렌드 · XML-RPC 즉시 발행만 존재
- **고도화 포인트**: 발굴(돈 되는 주제) · 성과·수익 매칭 · 예약/캘린더 · 학습 피드백
