# 1단계. 사용자 시나리오 (End-to-End)

> 블로그 글 작성을 자동화하고 **수익화**까지 닫힌 루프로 굴리고 싶은 사용자 관점에서 작성.

---

## 1.1 페르소나

**김수익 (35세, 직장인 N잡러)**

- 평일 저녁·주말에만 짬을 내서 네이버 블로그를 운영
- 목표: 애드포스트 + 쿠팡파트너스 + 체험단·협찬으로 **월 100만 원 자동 수익**
- 글재주는 있지만 **시간이 없음**
- 더 큰 문제: *"뭘 써야 돈이 되는지"* 와 *"내 글이 실제로 돈을 벌었는지"* 를 모름

### 핵심 페인 포인트

| 페인 | 결과 |
|---|---|
| 매번 "오늘 뭐 쓰지" 주제 고민에 30분 소모 | 시작도 못 함 |
| 열심히 썼는데 검색 노출이 안 됨 | 동기 상실 |
| 어떤 글이 수익을 냈는지 깜깜이 | 전략 없는 발행 반복 |
| 쿠팡 링크, 협찬 표기 매번 수작업 | 누락·실수 |
| 꾸준한 발행 리듬 못 만듦 | 블로그가 죽어감 |

---

## 1.2 한 줄 요약

> **"발굴 → 작성 → 최적화 → 발행 → 수익추적 → 학습"** 의 닫힌 루프를 자동으로 굴려,
> 김수익이 **검수·발행 외 시간은 거의 안 쓰면서도** 데이터 기반으로 수익을 키우는 도구.

---

## 1.3 End-to-End 여정 (7 Phase)

### 🟢 Phase 0 — 처음 셋업 (한 번만)

수익은 채널 연결에서 시작.

1. 네이버 블로그 계정 연결 (XML-RPC) — 다중 블로그·부계정 등록 가능
2. **과거 글 10개 자동 임포트** → 내 말투·문체 학습
3. **수익 채널 연결**: 애드포스트 ID, 쿠팡파트너스 파트너ID, 관심 협찬 카테고리
4. 운영 방향 설정: 주력 카테고리, 발행 횟수 목표, 톤(친근체/정보체)

### 🟢 Phase 1 — 발굴 ("돈 되는 주제 찾기")

앱을 켜면 **대시보드**가 먼저 "오늘의 추천 주제"를 띄움.

- **황금 키워드 추천**: 검색량 많음 × 경쟁 적음 → 점수화 정렬
- **수익성 신호**: 키워드별 예상 CPC + 제휴상품 매칭 여부 → "쿠팡 링크 붙이기 좋음" 표시
- **트렌드/급상승**: 데이터랩 + 블로그·카페 검색량으로 시즌·실시간 이슈 포착
- **경쟁 글 엿보기**: 상위 노출 글의 구성을 요약해 "이기는 구조" 제안

→ **[이걸로 글쓰기]** 버튼 한 번으로 Phase 2 진입.

### 🟢 Phase 2 — 글 생성

선택한 주제가 작성 화면에 컨텍스트로 자동 주입된 상태에서:

1. 음성 녹음 / 사진 / 메모 입력
2. AI가 **선택 키워드를 자연스럽게 녹인** SEO 구조의 초안을 스트리밍 생성
3. **수익화 자동 삽입**
   - 쿠팡파트너스: 본문에 맞는 상품을 찾아 제휴 링크 블록 제안
   - 협찬글이면 "내돈내산/협찬" 의무 표기 자동 삽입
4. 사진별 자동 캡션·alt 텍스트

### 🟢 Phase 3 — 검수·최적화

발행 전 **SEO/품질 점수판**:

- 제목·본문 키워드 밀도, 글자 수, 이미지 수, 가독성
- **네이버 저품질 유발 요소 경고**: 금지어·과도한 외부링크·중복 표현
- 추천 제목 A·B안, 추천 태그 (빅·미들·롱테일 균형)

### 🟢 Phase 4 — 발행·예약

- 즉시 발행 / **예약 발행** (예: 매일 오전 7시 골든타임)
- **발행 캘린더**: 한 주 콘텐츠를 미리 채워놓고 자동 발행
- 다블로그 동시·분산 발행

### 🟢 Phase 5 — 성과·수익 추적 ("깜깜이 탈출")

발행 후가 진짜 핵심. **글별 성과 카드**:

- 방문자·유입 키워드·검색 노출 순위 변화
- **수익 연결**: 애드포스트 수익 + 쿠팡 클릭·전환·정산액을 **글 단위로 매칭**
- **글별 ROI**: "작성 1시간 → 누적 ₩32,000" 형태로 시급 환산

### 🟢 Phase 6 — 수익화 관리

- 제휴 링크 성과 대시보드 (어떤 상품이 잘 팔리나)
- 협찬 제안·진행 상태 칸반 (제안받음 → 작성 → 발행 → 정산)
- 월별 수익 리포트 (채널별 합산)

### 🟢 Phase 7 — 학습·반복 (선순환)

- 성과 데이터를 Phase 1 추천에 피드백 → "잘 먹힌 주제·패턴" 학습 → 추천 정확도 상승
- 반자동 파이프라인: 추천 키워드로 **초안까지 자동 생성 → 검수 대기함** 적재
  → 사용자는 검수·발행만

## 1.4 현재 구현 상태 (Gap 분석)

| Phase | 구현됨 | 비어 있음 |
|---|---|---|
| 0. 셋업 | 네이버 인증, 과거글 폴더(`/posts`), AI 키 | 다블로그·수익채널 연결, 과거글 자동 임포트 |
| 1. 발굴 | — | **전부 비어 있음** |
| 2. 작성 | 음성/사진/메모/생성/스트리밍/말투학습 | 키워드 컨텍스트 주입, 쿠팡 자동 삽입, 협찬 표기 |
| 3. 검수 | 트렌드 분석(키워드/제목/태그) | SEO 점수판, 저품질 경고, 표절 검사 |
| 4. 발행 | XML-RPC 즉시 발행 | **예약·캘린더·검수 대기함** |
| 5. 추적 | — | **전부 비어 있음** |
| 6. 수익 | — | **전부 비어 있음** |
| 7. 학습 | — | **전부 비어 있음** |

→ 작성·일부 트렌드·발행만 됨. **발굴·성과·수익·예약·학습**이 통째로 비어 있음. 여기가 고도화 포인트.
