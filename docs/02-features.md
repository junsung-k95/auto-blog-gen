## Phase 1 — 발굴 (돈 되는 주제 찾기)

| # | 기능 | 우선순위 |
|---|---|---|
| 1.1 | **황금 키워드 추천 엔진** — 검색량/경쟁/CPC 스코어링 카드 리스트 | ⭐ |
| 1.2 | 키워드 데이터 수집: 블로그·카페 검색량, 데이터랩, 자동완성, 연관검색어 | ⭐ |
| 1.3 | 경쟁도 지표 — `블로그발행량/검색량` 비율 + 상위 노출 글 품질 | ⭐ |
| 1.4 | **수익성 점수** — 쿠팡 상품 매칭 가능 여부 + 애드포스트 단가 추정 | ⭐ |
| 1.5 | 실시간/시즌 트렌드 — 데이터랩 급상승, 주간 증가율 ≥20% 필터 | ⭐ |
| 1.6 | **카페 인기 토픽 스캐너** — 대형 카페 인기글 → 키워드 추출 | ⚪ |
| 1.7 | 경쟁 글 분석 — 상위 1~10위 글 구조 요약 + 이기는 구성 제안 | ⚪ |
| 1.8 | 키워드 저장함 / 무시 키워드 블랙리스트 | ⭐ |
| 1.9 | 카테고리별 추천 (내 운영 카테고리 우선 노출) | ⭐ |
| 1.10 | 매일 아침 추천 키워드 푸시/이메일 다이제스트 | ⚪ |
| 1.11 | 협찬 가능 키워드 표시 (체험단 사이트 크롤링 매칭) | 🔵 |
